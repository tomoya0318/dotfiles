  ■ 移動（カーソル）  ※数字を前に付けると回数指定（例 5j）
    h j k l         左 下 上 右
    Nj / Nk         N 行 下 / 上
    w / b / e       次の単語頭 / 前の単語頭 / 単語末
    0 / ^ / $       行頭 / 最初の文字 / 行末
    gg / G          ファイル先頭 / 末尾
    NG / :N         N 行目へジャンプ（例 42G）
    H / M / L       画面の 上 / 中 / 下 へ
    Ctrl-d / Ctrl-u 半ページ 下 / 上
    Ctrl-f / Ctrl-b 1ページ 下 / 上
    { / }           段落 前 / 後
    %               対応する括弧へ

  ■ 編集
    i / a           挿入（前 / 後）
    o / O           下 / 上に行を追加して挿入
    x / dd / yy     1字削除 / 行削除 / 行コピー
    p / P           貼り付け（後 / 前）
    Opt-j / Opt-k   行を 下 / 上 へ移動（選択中は範囲ごと）
    u / Ctrl-r      取消 / やり直し
    .               直前の操作を繰り返す
    〈文法〉操作+範囲: dw 単語削除 ・ ciw 単語変更 ・ di" 引用符の中

  ■ 選択（ビジュアル）
    v / V / Ctrl-v  文字 / 行 / 矩形 選択 → y コピー ・ d 削除

  ■ 検索
    /文字列 → n/N   前方検索 → 次 / 前
    *               カーソル下の単語を検索

  ■ 保存・終了
    :w 保存  :q 閉じる  :wq 保存して閉じる  ZZ 保存して閉じる
    Space q q       nvim ごと終了
